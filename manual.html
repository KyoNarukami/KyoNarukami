USER'S GUIDE

To insert a plate number, the user must log in into the Local Host Webpage of the device. The user will then input the plate number of the device they're giving access to.

In order to Activate the gate, the Camera should get a clear view of the plate number, and then Just wait for 5-10 seconds for the processing and analyzing of the image. The Gate then will open once it confirms that it matches the Input in the webpage.
